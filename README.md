## Instalasi

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser di [http://localhost:3000](http://localhost:3000)

## Struktur Project

- `/app` - Halaman dan routes (App Router)
- `/app/api` - API routes untuk CRUD operations
- `/components` - Komponen reusable
- `/lib` - Utilities dan types

## Teknologi

- Next.js 14
- React 18
- TypeScript
- CSS Modules
